Acunetix Plugin for Acunetix Enterprise Edition
<br>
<br>
Required configuration fields:<br>
- 'apiKey' The Acunetix API Key genenerated in the Acunetix Enterprise Edition<br>
- 'scanProfileId' The Scan Profile ID Available in the Acunetix configuration<br>
- 'reportTemplateId' The Report Template ID Available in the Acunetix configuration<br><br>
In order to speed up information calls direct database calls are required<br>
- 'dbName' The Acunetix database name<br>
- 'dbUser' The Acunetix database user name<br>
- 'dbPassword' The Acunetix database password<br>
- 'dbHost' The Acunetix database host<br>
- 'dbPort' The Acunetix database port<br>
