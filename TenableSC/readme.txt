<H1>Tenable Security Center Plugin</H1>
<br>
<H2>Required configuration fields:</H2><br>
- <b>username</b><br>
- <b>password</b><br>
- <b>host</b><br>
- <b>port</b><br>
- <b>copyScanUserId</b> - The User ID of a scan that can be copied and reran with different parameters<br>
- <b>copyScanName</b> - The name of the scan that can be copied and reran with different parameters<br>
- <b>copyScanId</b> - An ID of a scan that can be copied and reran with different parameters<br>
- <b>excludedPlugins</b> - A comma sepparated list of plugin IDs to exclude from the scans
